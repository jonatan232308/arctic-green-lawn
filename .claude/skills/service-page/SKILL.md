# /service-page — Arctic Green Hugo zipper-page generator

Generates a service×city landing page targeting a single money keyword. Output goes in `content/service-areas/<service>-<city>.md`.

## When to invoke

- User: `/service-page lawn mowing cypress tx`
- User: "build me a service page for tree trimming the woodlands"
- After SEMrush keyword validation, batch-build top 10 winners

## Inputs needed

1. Primary keyword (e.g., `lawn mowing cypress tx`)
2. Service from list: lawn-mowing-maintenance, spring-fall-cleanup, landscape-design-consultation, irrigation-system-check, tree-shrub-trimming, hardscape-paver-walkways, free-in-home-consultation
3. City from list: cypress, tomball, spring, the-woodlands, magnolia, katy

## Pipeline

1. **Read voice files** from `.claude/seo/voice/`:
   - voice.md — tone rules
   - humor.md — joke cadence + reference passages
   - stats.md — real numbers to cite
   - stories.md — anecdotes to inject (pick 1)
   - opinions.md — strong takes (pick 1)

2. **Steal-the-SERP** — fetch top 3 ranking pages for the keyword (excl. Reddit/YouTube/forums). Use WebFetch or Firecrawl. Extract:
   - Average word count
   - H2 count + topics covered
   - Image count
   - Local trust signals used (years in biz, reviews, area-specific info)

3. **Build frontmatter**:
   ```yaml
   ---
   title: "[Service] in [City], TX"
   description: "[150-160 chars, primary keyword in first 100, phone, value prop]"
   service: "[service slug]"
   city: "[city slug]"
   keywords: ["[primary]", "[secondary 1]", "[secondary 2]"]
   neighborhoods: [list 4-6 from city's areas page]
   zipcodes: [from stats.md]
   date: YYYY-MM-DD
   ---
   ```

4. **Body structure** (target average word count from SERP analysis, typically 800-1500):
   - **Opening** (50 words): primary keyword in first 100 words, ONE human moment from humor.md, hook tied to opinions.md
   - **H2: Why [city] homeowners hire Arctic Green for [service]** — neighborhood names, zip codes, locally-owned angle
   - **H2: What's included in [service] in [city]** — bullet list, specific to the service
   - **H2: [city]-specific notes** — soil/grass/weather quirks for that area
   - **H2: How pricing works** — direct $ if available, "varies by yard size" honest
   - **H2: Frequently asked questions** — 4-6 H3 questions (steal from People Also Ask)
   - **CTA block**: Phone (903) 658-8244 + form link

5. **On-page SEO checklist** (apply ALL):
   - [ ] Primary keyword in title, H1, first 100 words, meta description
   - [ ] Exactly 1 H1 (the title)
   - [ ] 4-8 H2/H3 headings
   - [ ] 4-6 internal links: link to service page (`/services/<slug>/`), city page (`/areas/<city>/`), 2 related zipper pages, 1 blog post
   - [ ] 2-3 external links: city government site, weather/soil reference, grass-type extension service
   - [ ] Alt text on all images, primary keyword in 1 alt
   - [ ] Schema.org LocalBusiness JSON-LD (use Hugo partial — see template)
   - [ ] Phone number in clickable `tel:` link
   - [ ] No keyword stuffing — natural density

6. **Inject ONE anecdote** from stories.md if any matches. If none match, skip — don't fabricate.

7. **Inject ONE opinion** from opinions.md as a callout box.

## File output

`content/service-areas/<service>-<city>.md`

If `content/service-areas/` doesn't exist, create it + add `_index.md` index page.

## Hugo build check

After writing, run:
```bash
cd ~/arctic-green-lawn && hugo --quiet --buildDrafts=false 2>&1 | tail -20
```

If errors, fix before reporting done.

## Cadence guard

Per Jono's framework: never publish more than 5 zipper pages per day. Stagger 1-2/day for first 2 weeks. Track in `.claude/seo/published-log.md`.

## Reference

Top of every output file, comment line:
```
<!-- Generated by /service-page skill. Voice: see .claude/seo/voice/. Validate Lighthouse 100/100 before push. -->
```

## Hard rules

- Never use trigger phrases from voice.md ("comprehensive guide", "in today's fast-paced world", etc.)
- Never fabricate stats — only use values from stats.md
- Never name competitors negatively
- Never claim certifications/awards Arctic Green doesn't have
- Always include phone (903) 658-8244 + form link in CTA
